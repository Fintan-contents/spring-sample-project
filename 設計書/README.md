# サンプルプロジェクト設計書

以下の構成でサンプルプロジェクトの設計書を格納している。

```
.
├── A1_プロジェクト管理システム
│   ├── 010_要件定義
│   │   └── 040_画面設計
│   ├── 020_方式設計
│   │   └── 020_開発標準
│   │       ├── 010_設計標準
│   │       └── 020_テスト標準
│   ├── 030_アプリ設計
│   │   ├── 010_システム機能設計
│   │   ├── 020_共通コンポーネント設計
│   │   ├── 030_インタフェース設計
│   │   ├── 050_メッセージ設計
│   │   ├── 060_コード設計
│   │   ├── 070_データモデル設計
│   │   ├── 100_ジョブフロー設計
│   │   └── 110_テスト仕様書
│   ├── 080_ツール
│   └── 100_レビュー記録
└── B1_顧客管理システム
    └── 030_アプリ設計
        ├── 010_システム機能設計
        ├── 030_インタフェース設計
        ├── 050_メッセージ設計
        ├── 060_コード設計
        ├── 070_データモデル設計
        └── 110_テスト仕様書
```
